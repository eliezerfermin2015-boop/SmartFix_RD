# SmartFix RD

## Descripción
SmartFix RD es un sistema web para la gestión de talleres de reparación de equipos electrónicos.

## Objetivo
Facilitar la administración de clientes, inventario y órdenes de reparación mediante una aplicación web.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- LocalStorage
- Git
- GitHub

## Instalación
1. Descargue el proyecto.
2. Abra la carpeta.
3. Ejecute `index.html` en su navegador.
